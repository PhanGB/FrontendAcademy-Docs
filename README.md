# 🎓 Frontend Academy

> Một nền tảng học Frontend toàn diện được xây dựng bằng **Laravel 12 + Bootstrap 5**, hướng đến việc học theo lộ trình, thực hành qua dự án thực tế và phát triển theo chuẩn kỹ sư phần mềm.

---

# 📖 Giới thiệu

Frontend Academy là dự án xây dựng một nền tảng học Frontend miễn phí với mục tiêu giúp người học có thể tiếp cận kiến thức từ cơ bản đến nâng cao theo một lộ trình rõ ràng.

Không chỉ là nơi đọc tài liệu, Frontend Academy còn hướng đến việc tạo ra một môi trường học tập có tính tương tác cao với:

- Bài học trực quan
- Ví dụ thực tế
- Quiz kiểm tra kiến thức
- Coding Exercise
- Project thực hành
- Theo dõi tiến độ học
- Hệ thống thành tựu
- Chứng chỉ hoàn thành khóa học

Mục tiêu cuối cùng là xây dựng một nền tảng có trải nghiệm tương tự:

- freeCodeCamp
- Codecademy
- MDN Web Docs
- W3Schools

# 📂 Cấu trúc tài liệu


Thứ tự đọc được khuyến nghị:

# 🏗 Kiến trúc dự án

Dự án áp dụng triết lý:

> **Documentation First Development**

Mọi quyết định về kiến trúc đều được tài liệu hóa trước khi viết mã nguồn.

Quy trình phát triển:

```text
Tài liệu
    ↓
Kiến trúc
    ↓
Thiết kế Database
    ↓
Quy chuẩn Coding
    ↓
Thiết kế UI
    ↓
Xây dựng Component
    ↓
Lập trình
    ↓
Kiểm thử
    ↓
Triển khai
```

# 🤖 Hướng dẫn dành cho AI Coding Agent

Dự án được thiết kế để có thể phát triển cùng AI (ChatGPT, Codex, Claude, Gemini...).

Nếu bạn là AI Coding Agent, hãy thực hiện theo thứ tự:

1. Đọc `AGENTS.md`.
2. Đọc `README.md`.
3. Đọc toàn bộ thư mục `docs/`.
4. Hiểu kiến trúc trước khi viết code.
5. Tuân thủ Coding Standard.
6. Không thay đổi kiến trúc nếu chưa được yêu cầu.
7. Tái sử dụng Component hiện có.
8. Cập nhật tài liệu nếu có thay đổi kiến trúc.
